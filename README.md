**BlackJack korttipeli projekti**

Tavoitteenani on luoda toimiva BlackJack korttipeli käyttäen Java ohjelmointikieltä.
Blackjack on perinteinen korttipeli, jossa pelataan pääsääntöisesti jakajaa vastaan.
